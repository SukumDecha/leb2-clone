CSS Basics

1. object-fit

Property object-fit ใช้กำหนดวิธีการปรับขนาดของวัตถุภาพภายใน Element โดยสามารถให้ภาพขยายหรือย่อให้พอดีกับขนาดของ Element ได้

2. linear-gradient

Property linear-gradient ใช้สร้างพื้นหลังที่มีการเปลี่ยนแปลงสีตามเส้นทางเชิงเส้น โดยสามารถกำหนดสีหลายสีในการสร้าง Gradient

3. white-space

Property white-space ใช้กำหนดการจัดวางของข้อความภายใน Element โดยสามารถปรับเปลี่ยนการจัดวางข้อความแบบ Wrap หรือ No Wrap ได้ [Example]

4. filter

Property filter นำไปใช้เพื่อใส่เอฟเฟกต์ทางภาพ เช่น เบลอหรือการเปลี่ยนสีขององค์ประกอบใน HTML element ที่ระบุ

5. transition

Property transition นำไปใช้เพื่อกำหนดการเปลี่ยนแปลงของสมบัติต่าง ๆ ขององค์ประกอบใน HTML โดยให้เกิดอย่าง smooth เป็นธรรมชาติ.

6. calc

Property calc เป็นฟังก์ชันใน CSS ที่ใช้ในการคำนวณค่าในส่วนต่าง ๆ ของคุณสมบัติที่ระบุค่าความกว้าง (width), ความสูง (height), ขนาดตัวอักษร (font-size) และอื่น ๆ ตามเงื่อนไขต่าง ๆ ที่กำหนดให้ได้ผลลัพธ์ที่ต้องการ

7. brighness

Property brighness Brightness ปรับตวามสว่างของภาพ ค่าอยู่ระหว่าง 0 (มืด) ถึง 1 (สว่าง)
